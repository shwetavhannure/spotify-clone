# Spotify Clone – Frontend

Spotify Clone is a responsive web music player interface inspired by the official Spotify Web Player.

This project recreates the layout and user interface of Spotify using HTML and CSS. It includes a sidebar navigation menu, music player controls, playlist cards, and a responsive layout designed to resemble the real Spotify interface.

------------------------------------------------------------

## 🚀 Features

🎵 Spotify-inspired user interface  
📂 Sidebar navigation with Home, Search, and Library sections  
🎧 Music player section with playback controls  
📑 Playlist and trending music cards  
📱 Responsive layout using CSS media queries  
✨ Interactive hover effects for icons and cards  

------------------------------------------------------------

## 🛠 Tech Stack

HTML5  
CSS3  
Flexbox  
Google Fonts (Montserrat)  
Font Awesome Icons
